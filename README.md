# 🔐 Password Strength Analyzer

A Python-based tool to evaluate password strength and identify weak patterns.

---

## 🚀 Features
- Checks password length
- Detects uppercase, lowercase, numbers, special characters
- Provides strength rating (Weak / Medium / Strong)
- Gives feedback for improvement

---

## ⚙️ Tech Used
- Python

---

## ▶️ How to Run
```bash
python password_analyzer.py
